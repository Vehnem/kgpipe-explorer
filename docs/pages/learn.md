# Learn page

**UI:** `frontend/src/pages/LearnPage.tsx`

## Supports

- Welcome / hero and suggested workflow cards (Editor, Results, Leaderboard)
- Metadata Explorer callout for debugging underlying graph metadata
- References (demo repo, KGpipe, papers)
- Knowledge Graph primer cards (KG, triples, RDF, SPARQL, pipelines, metrics/rankings)
- Note on **export → external run → reimport** before exploring results

## Does not

- Call the backend (works even if `/tasks` fails)
- Execute pipelines
